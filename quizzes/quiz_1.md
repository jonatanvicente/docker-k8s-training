

**1. What is a main difference between containers and virtual machines?**  
a) Containers require a hypervisor, VMs do not  
b) Containers share the host OS kernel, VMs include a full OS  
c) Containers are always slower than VMs  
d) VMs cannot run applications  

<details>
  <summary>Solution</summary>

</details>
